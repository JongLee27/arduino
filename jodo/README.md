# 조도센서를 통해 식물 생장 LED 제어하기

## 조건
조도센서 측정 범위 : 49 ~ 969(어두울수록 값이 커지고 밝을수록 값이 작아짐)

-> 어두울 때의 값을 400을 기준으로 하여 400 이상일 경우 : **식물 생장 LED를 켬**

-> 밝을 때의 값을 400을 기준으로 하여 400 이하일 경우 : **식물 생장 LED를 끔**
