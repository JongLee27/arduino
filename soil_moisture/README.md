## 진행 사항
- 토양 수분 감지 센서, 워터펌프 모두 작동 확인
- 토양 수분 감지 센서를 작동해본 결과 물에 아예 담가놓아도 값이 800 내외로 잡힘
  - 이 부분은 일단 아날로그 값 매핑을 800 ~ 1023로 진행
- 테스트할 때는 워터 펌프를 5초간 작동하게 설정해놓았는데, 실제 토양에서는 몇 초를 작동해야 토양의 수분이 올라가는 지 테스트해봐야 함
