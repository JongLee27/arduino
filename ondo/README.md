# 아두이노와 온습도 센서(DHT-11)을 통해 온도 및 습도 측정하기

# 온습도 센서 연결
1. 온습도 센서를 아두이노에 연결해줍니다.
2. 왼쪽부터 5V, 2, GND에 연결합니다.

# 온습도 센서 라이브러리 설치
1. 아두이노 IDE를 열고, 상단 메뉴에서 "스케치"를 선택합니다.
2. "라이브러리 포함하기" -> "ZIP 라이브러리 추가"를 클릭합니다.
3. "DHT11_library.zip" 파일을 찾아 추가합니다.
